# Migration — v5.0.1

No mandatory project-file migration.

This patch corrects installed workflow behavior:

- pinned websites without an explicit human-selected concept enter provided-reference concept discovery;
- both concept-discovery paths generate/refresh the Project Dashboard and stop for human selection;
- Round 2 requires a complete human selection record in `brain/research/concepts.md`;
- Build Page rejects claims of research-grounded composition when existing research has no explicit selection.

Existing research, moodboards, teardowns, BRAND, DESIGN, and page code are preserved. Projects with an agent-inferred selection may re-enter UI Research at the human checkpoint; migration does not rewrite their canon automatically.
