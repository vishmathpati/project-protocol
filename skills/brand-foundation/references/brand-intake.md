# Brand intake reference

Read BRIEF, STRUCTURE, BRAND, WONT-DO, and TASTE first. Build an internal map of what is already known. Ask only gaps that affect brand differentiation, emotional transformation, voice, cultural context/scripts, trust, memorability, distinctive/protected assets, reference tribe, or refusals.

Accept free-form answers. Preserve the user's exact language as evidence, then synthesize it into the BRAND structure. Never ask project identity, users, problem, business, stage, parts, or scope when Foundation already owns them. Never ask colors, fonts, pages, layouts, research depth, or implementation.
