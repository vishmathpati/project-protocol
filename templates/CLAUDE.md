# [Project Name]

## Constitution

1. Read live project sources before answering about project state.
2. Human explicitly invokes CEO, worker, or solo; never guess authority.
3. Chapter scope governs worker authority inside an isolated worktree.
4. Worker branches are proposals; only CEO approval and merge establish canon.
5. Existing project configuration and design-system components win.
6. Ask only for decisions that cannot be discovered from the repository or canon.
7. Save is persistence, not completion, approval, or merge.
8. Stage explicit owned files; preserve unrelated work.
