## Optional business-specific registries

Add only useful sections such as locations, rooms, treatments, menu categories, integrations, plans, comparison facts, events, or case studies. Do not force a SaaS feature table onto every project.

## Editing rules

- Stable IDs are renamed through an intentional migration once consumed by live pages.
- One primary conversion per page; supporting CTAs may exist.
- Claims and media retain source/rights status.
- Clearly labeled demo data is allowed only for software screenshots; never use it as customer proof.
