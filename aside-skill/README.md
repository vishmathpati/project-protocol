# Aside skills

Standalone browser skills uploaded into Aside, not loaded as Claude Code/Codex plugin skills.

| Aside skill | Project Protocol driver | Purpose |
|---|---|---|
| `ui-research/SKILL.md` | `ui-research` | Sweep a niche into concepts, then deeply tear down the selected concept. |
| `inspect-component/SKILL.md` | `inspect-component` | Investigate one exact external UI region and return implementation evidence. |

Upload each skill once and re-upload when its interface changes. Project Protocol supplies project-specific missions and ingests the returned evidence into Markdown canon. Aside does not approve design decisions or write implementation code.
